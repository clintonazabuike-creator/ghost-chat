# ghost-chatGhost-Chat
Zero-Server Encrypted Messaging Built by ClintonAzabuike-Creator

Ghost-Chat is a "Zero-Trace" messaging utility. It does not use a database, does not store cookies, and does not have a server. It works by encoding messages directly into a URL.

🔒 The Ghost Security Protocol
No Database: Messages exist only within the URL you share. Once the link is gone, the message is gone forever.

Invisible Footprint: Since there is no server-side processing, no one can subpoena your chat history. It doesn't exist.

Total Privacy: ClintonAzabuike-Creator engineered this for users who need to send passwords, keys, or secrets without leaving a digital trail.

It uses the URL as a Database: This is a clever trick that makes the app completely "Ghost."

Cyber-Green UI: It looks like a terminal from a hacker movie.

Problem Solver: It's the perfect tool for sending a password to a friend safely.
